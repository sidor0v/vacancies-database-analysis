# Проект "Анализ вакансий hh.ru"

Данная работа выполнена в рамках учебного курса по Data Science и состоит из следующих частей:
* предварительный анализ данных;
* анализ вакансий;
* анализ работодателей;
* предметный анализ (вакансии по направлению Data Scence).

Все данные для проекта были получены посредством sql-запросов к базе Metabase образовательной организации. Соединение с базой выполнялось с помощью библиотеки psycopg24. Из соображений конфиденциальности **пароль для установки соединения с базой данных был удален из проекта** перед загузкой на github.

Проект представлен [ноутбуком](https://github.com/sidor0v/vacancies-database-analysis/blob/main/hh-part2.ipynb). Визуализация данных выполнена с помощью библиотеки Plotly. Для корректного отображения графиков и диаграмм на GitHub ноутбук сохранен со статическими (не интерактивными) визуализациями. Однако ноутбук можно скачать и изменить значение глобальной переменной *px_mode* на пустую строку:  ```px_mode = ''```. После этого выполнить код и получить интерактивные диаграммы и графики Plotly.

Дополнительные исследования данных в проекте выполнены после обязательных, предусмотренных заданием пунктов в каждом разделе проекта.

Выводы в проекте сделаны по результатам обязательных и дополнительных исследований.
